# Nekton Admin

## 项目用途

Nekton Admin 是一个企业级多租户管理系统，专为需要为多个客户提供独立管理平台的服务商或企业设计。该系统提供了一个统一的后台管理框架，支持为不同客户（租户）创建完全隔离的管理环境，每个租户拥有独立的用户体系、权限管理和业务数据。

## 适用场景

- **SaaS服务平台**：为多个客户提供独立的管理后台
- **集团企业管理**：为不同子公司或部门提供独立管理系统
- **多品牌运营**：为不同品牌提供独立的管理平台
- **系统集成商**：为客户提供定制化的管理解决方案

## 核心功能

### 1. 多租户管理
- 支持创建和管理多个租户
- 每个租户拥有独立的数据库和用户体系
- 租户间数据完全隔离，确保数据安全

### 2. 用户与权限管理
- 灵活的角色和权限配置
- 基于角色的访问控制（RBAC）
- 可自定义菜单权限，实现精细化权限管理

### 3. 工作流引擎
- 可视化工作流定义和管理
- 支持与外部系统（如金蝶等ERP系统）集成
- 灵活的工作流节点配置和条件判断

### 4. 系统集成
- 提供标准API接口，支持第三方系统集成
- 支持多种数据库连接和动态切换
- 可扩展的插件架构，便于定制开发

## 快速体验

### 环境要求
- JDK 17+
- PostgreSQL 10+
- Maven 3.6+

### 启动步骤
1. 配置数据库连接信息
2. 执行数据库初始化脚本
3. 运行应用：`mvn spring-boot:run`
4. 访问地址：`http://localhost:8080/api`

## 项目优势

- **多租户隔离**：确保不同客户数据完全隔离，提高安全性
- **灵活权限**：支持细粒度权限控制，满足不同业务需求
- **易于扩展**：模块化设计，便于功能扩展和定制开发
- **工作流支持**：内置工作流引擎，支持复杂业务流程管理
- **系统集成**：提供丰富的API接口，便于与其他系统集成
